# Ubuntu 基础与入门

## Terminal

您可能需要经常使用 `终端` 执行命令。

您可以在 Dash 中找到 Terminal 应用打开终端。

您也可以按 `Ctrl + Alt + T` 打开一个终端。

## 执行 `.sh` 文件

在 Ubuntu 上下载可执行文件后，您无法立刻执行它。

您需要执行以下指令改变文件的 **权限**

    $ chmod +x install.sh

然后执行 `install.sh`

    $ ./install.sh

## 翻墙

当您查阅关于 Ubuntu 的技术资料时，您不可避免地需要使用 Google 等境外服务。

您可以自购 Shadowsocks, VPN 或是 SSLedge 翻墙。
